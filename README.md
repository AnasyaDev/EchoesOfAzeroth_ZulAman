# Echoes of Azeroth: Zul'Aman

`EchoesOfAzeroth_ZulAman` is a troll-focused content plugin for `EchoesOfAzeroth`.

It maps Zul'Aman itself and also ships a broad selection of troll-themed music packs that can be assigned manually elsewhere through the core addon UI.

## Requires

- `EchoesOfAzeroth`

This plugin depends on the core addon.

## Included music packs

The plugin includes:

- `Zul'Aman (TBC)`
- `Zul'Aman (Midnight)`
- `Zul'Aman (TBC + Midnight)`
- `Amani`
- `Darkspear`
- `Drakkari (Northrend)`
- `Gurubashi (Stranglethorn)`
- `Sandfury (Tanaris / Zul'Farrak)`
- `Zandalari (Pandaria)`
- `Zandalari (BFA)`
- `Atal'ai`
- `Troll Mixed`

## Covered zone

The plugin currently maps:

- Zul'Aman

The other packs are intentionally available for manual assignment through `Zone Mapping` and `Music Packs`.

## Source approach

The track catalog combines:

- original WoW zone music
- later expansion troll-themed tracks
- Midnight troll and Amani material

Track durations were measured from the downloadable audio files so pack rotation can follow real track length.

## How to use it

1. Install `EchoesOfAzeroth`
2. Install this plugin
3. Open `/eoa options`
4. Enable the plugin in `Plugins`
5. Use `Zone Mapping` or `Music Packs` to assign the troll packs you want

## Notes

- The plugin provides content only
- Playback, profiles, custom packs, custom zones, and import/export live in the core addon
- Installing this plugin through CurseForge should also install the core addon when project relations are configured as required dependencies

## Community profiles

`EchoesOfAzeroth` is meant to grow as a community addon.

If you build a profile that feels especially polished, immersive, or lore-accurate, please share it. Great community profiles can help refine the addon over time, and an especially strong configuration may be adopted as an official default or recommended setup in the future.
